Introduction

MarkdownPlus is a transpiled language for writing documentations using the Gyro language.

It was implemented in JavaScript and is ideal for creating API documentation, tutorials and other material. MarkdownPlus is fully customizable and has been written with simplicity in mind.

  • A lightweight Markdown syntax helps you focus on your content.
  • Theme files let you fully customize the output.
  • Providers can deliver custom syntax and modify the content to your liking.
  • Backends allow you to render the output to multiple languages.
  • MarkdownPlus was written in JavaScript for a huge backlog of possibilities.
  • Playgrounds let your users experience your documentation interactively.
  • Search allows for finding your content with ease.
  • Automatically Resolving Links let you write faster.

This guide will lead you through the intrinsics of MarkdownPlus and show you what it's default settings produce.
